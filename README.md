# 💫 About Me:
Im currently working on my main project EH-Stats which is a Discord-Bot with Dashboard that is mainly used by Roleplay-Servers (especially Emergency-Hamburg Roleplay Servers) and provides features such as Monitoring, Shift-MAnagement and more. ALos im currently working on Aegis-Guard, which is a Security-Bot.
